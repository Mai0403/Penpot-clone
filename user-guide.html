Penpot User Guide

In this documentation you will find (almost) everything you need to
know about how to work with Penpot, from the interface basics to
advanced functionality. Learn the details on topics such as
prototyping, organizing your designs, or sharing, to get the most
out of Penpot.

This documentation is a work in progress that will be updated frequently. If you
have a suggestion, see something is missing or find anything that needs correcting,
please write to us at support@penpot.app.
